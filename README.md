# 📄 Image to PDF Converter

A lightweight, fast, and secure web application that converts JPG and PNG images into a single PDF file. 

The best part? **Everything runs locally in your browser.** No files are uploaded to any server, ensuring total privacy for your documents.

✨ **[Check out the Live Demo here!](https://seu-usuario.github.io/img-to-pdf/)** ✨

---

## 🚀 Features

- **100% Client-Side:** Processing is done entirely in the browser using JavaScript.
- **Privacy First:** Your images never leave your computer.
- **Auto-Scaling:** Images are automatically resized and centered to fit perfectly on standard A4 pages without distortion.
- **Multi-File Support:** Select multiple images at once to merge them into a single PDF document.
- **Responsive Design:** Clean, GitHub-styled minimalist interface.

---

## 🛠️ Built With

* **HTML5 & CSS3:** For the structure and styling.
* **Vanilla JavaScript:** For the core logic and asynchronous file handling.
* **[jsPDF](https://github.com/parallax/jsPDF):** A reliable client-side library for generating PDFs.
